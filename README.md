# Automatic Door Indicator using IR Sensor

This project is a smart automatic door indicator system made using Arduino and an IR sensor in Tinkercad.

## Working
- When a person/object is detected by the IR sensor:
  - GREEN LED turns ON (Door Open)
  - RED LED turns OFF

- When no object is detected:
  - RED LED turns ON (Door Closed)
  - GREEN LED turns OFF

## Components Used
- Arduino Uno
- IR Sensor
- Green LED
- Red LED
- Jumper Wires

## Software Used
- Tinkercad
- Arduino IDE
